# Welcome to Rill

This is a sample note. Feel free to delete it once you get the feel of things.

Rill turns voice and text journals into atomic knowledge. There are three primitives:

- **Capture** — Jot down journals, meetings, and web clips into `inbox/`
- **Distill** — Run `/distill` in Claude Code to extract atomic notes, tasks, and entities from what you captured
- **Browse** — Review what Rill has accumulated across this GUI's views (Timeline, Knowledge, Tasks, Workspaces, Pages, Reports)

If you want a guided tour, try `/onboarding` in Claude Code. To start capturing, click **New Entry** in the Timeline view or tell Claude "record this as a journal".
